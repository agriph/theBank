# theBank
#### An API for a bank where you can create an account, make transfers as well as transactions.

### Information / Introduction
This Bank API is a school project for a course in systematic software development. It can be used to create bank accounts that has an optional inital sum, which can then be used to make transfers and transactions that can also be deleted or altered. Each account has a balance overview that can also be manually updated.

### Technologies
This API has been made using Eclipse (Java), Spring Framework, Swagger UI, Postman and SQL Server Management Studio.
